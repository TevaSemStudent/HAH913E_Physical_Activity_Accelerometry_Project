#  Data

## Acceleration data:

All accelerationdta was exported as csv files following the format date,x,y,z. No header are present in the files.

- 23_10_25_raw_cut.csv : acceleration data recorded on October 23rd in raw sensor units (1/256 g) during several jump sequences. Please note that the acceleration along the X and Y axis exceed the sensor's range, which was set at 8g.
- 26_10_25_raw_cut.csv :  acceleration data recorded on October 26th in raw sensor units (1/256 g) during several jump sequences. Please note that the acceleration along the X and Y axis exceed the sensor's range, which was set at 8g, and that there is significant noise on the signal.
- 24_11_raw.csv : acceleration data recorded on November 24th in raw sensor units (1/256 g) during three jump sequences. Please note that the acceleration along the Y axis exceed the sensor's max range, which was 16g.
- 24_11_g.csv : acceleration data recorded on November 24th in g during three jump sequences. Please note that the acceleration along the Y axis exceed the sensor's max range, which was 16g.
